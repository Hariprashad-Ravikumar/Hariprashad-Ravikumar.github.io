I am a PhD Candidate at New Mexico State University specializing in High-Performance Computing (HPC) and Deep Learning, with deep expertise in GPU-accelerated computing using C++/CUDA and Python. My work involves developing and optimizing software for large-scale data analysis and scientific simulation on supercomputing clusters.

My doctoral research focuses on applying advanced computational techniques to solve complex problems in particle physics. Key technical achievements include:

  • Performance Engineering & Acceleration: I developed GPU-accelerated CUDA C++ pipelines that reduced data processing time by 10x for multi-terabyte Fourier transforms on HPC clusters. As part of my collaboration with Los Alamos National Laboratory, I design and optimize parallelized C++ CUDA kernels that significantly accelerate multi-terabyte Monte Carlo simulations on clusters like the NERSC Perlmutter.
  • End-to-End Machine Learning Pipelines: I built a complete ML pipeline to process over 30,000 multi-dimensional observables from simulations. Using symbolic regression (PySR) with custom, physics-constrained loss functions, this pipeline achieved a model fit accuracy of over 93%.
  • Large-Scale System Orchestration: I have designed and deployed custom SLURM workflows to manage and execute over 75,000 CPU/GPU compute hours, enabling robust, automated parallel analysis for large-scale jobs.
  • Robust Software Development & Validation: I created production-grade Python and Mathematica packages to ensure numerical stability and reproducibility in multi-stage data analysis workflows. To increase model reliability, I apply rigorous validation methods, including AIC-based selection, chi-squared minimization, and bootstrap/jackknife resampling across tens of thousands of correlated data points.

In addition to my core research, I actively pursue independent projects to broaden my technical skill set. I built a cloud-hosted ML forecasting platform using AWS/Azure, Flask, and React, featuring automated data pipelines and a GPT API integration for generating natural-language summaries. I have also implemented neural networks from scratch in NumPy to solidify my understanding of fundamental ML principles.

With a strong foundation in C++, Python, CUDA, and parallel computing, complemented by hands-on experience in MLOps, cloud technologies, and rigorous data analysis, I am passionate about building efficient, scalable, and impactful software solutions for data-intensive challenges.
